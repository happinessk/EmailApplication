# EmailApplication

The goal of this app is to: create email accounts

The application does the following:
1. Generates an email with the following syntax: firstname.lastname@department.company.com
2. Determines the department (sales, development, accounting), if none leave blank
3. Generates a random String for a password
4. Has a set methods to change the password, set the mailbox capacity, and define an alternate
email address
5. Has a get methods to display the name, email, and mailbox capacity
